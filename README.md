## Install
Clone this project

```bash
> https://github.com/ground67/WHAT-BOT.git
> cd WHAT-BOT
```

Install the dependencies:

```bash
> bash install.sh
```

### Usage
Run the Whatsapp bot

```bash
> npm start
```
